  VPulse is a lightweight uptime tracker for Windows systems that logs and analyzes your device usage over time, helping you understand your daily and monthly activity.
</div>

> [!WARNING]
> VPulse is currently under active development and not ready for daily use. The application requires manual setup and execution via the command line.

> [!NOTE]
> A more user-friendly version (GUI) is planned for future releases.

## ✅ Features
- Track system uptime

## ▶️ How to Run
1. Clone this repository
   ```bash
   git clone https://github.com/vrw93/VPulse.git
   ```
2. Install dependencies:
   ```bash
   pip install psutil
   ```
3. Run the script
   ```bash
   python main.py
   ```
